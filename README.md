### подготовка

создать `.env` с содержимым из `.env.example`

### установить зависимости

```bash
npm i
```

### поднять окружение

```bash
make up
```

### запустить миграции

```bash
make migrations-up
```

### сидировать базу

```bash
make seed-up
```

### запустить

```bash
npm run start:dev
```
